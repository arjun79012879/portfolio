My Projects

Fake New Detections

This project focuses on developing a system to automatically detect fake news articles using machine learning and natural language processing (NLP) techniques. By analyzing text data for linguistic patterns, sentiment, and source credibility, the system classifies news as real or fake. Various machine learning algorithms such as SVM, Random Forest, and Logistic Regression are evaluated alongside deep learning models like CNNs and RNNs. The model is built using Python and deployed with a user interface for real-time news verification. The system aims to combat misinformation and support informed public discourse.
